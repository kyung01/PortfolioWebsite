Reflective Shadow Map

DirectX based demo showcasing reflective shadow maps with deferred rendering process. Done as a solo project for Found of Game Graphics Program class at Rochester Institute of Technology.

Fractionauts

A rocket themed educational game targeted towards 4th grade students interested in learning how to perform basic arithmetic with fractions. Done as a group project for Humanitarian FOSS Development class at Rochester Institute of Technology.

Two Puzzle Games For Mobile And PC

First puzzle game is basically copy of game called "Endings". I have copied 50+ maps in the original game exactly both in terms of design and logic behaviour of enemies. Second game is my own creation. It combined turn based action puzzle concept from "Endings" with Galaxy like gameplay.

Four Android Games

These are early rough projects in which I attempted in order to learn Android. They are side scrolling shooter, flood puzzle, connecting dots simple mindless time killer, and more complex shape cutting puzzle game. All of them are copies of inspiration. They helped me to get started.
